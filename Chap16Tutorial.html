16.2.4. Challenging BQH

During the day, there are busy times when many flights depart and times when fewer flights depart.

Make a graph to show the distribution of flight departure time on January 01, 2013 using R code. Use 5 minutes as the frequency.

## [1] "Put the departure year, month, day, time data in the date_time format first, then make a graph with the date_time items.\n\nmake_datetime_100 <- function()\nflights_dt <- flights %>% \n  mutate()\nflights_dt %>% \n  filter()\n  ggplot() + \n  geom_freqpoly(binwidth = ) \nbinwidth determines the frequency in second (e.g. 10 minutes mean binwidth = 600)\n"
make_datetime_100 <- function(year, month, day, time) {
  make_datetime(year, month, day, time %/% 100, time %% 100)
}

flights_dt <- flights %>% 
  mutate(
    dep_time = make_datetime_100(year, month, day, dep_time))

flights_dt %>% 
  filter(dep_time < ymd(20130102)) %>% 
  ggplot(aes(dep_time)) + 
  geom_freqpoly(binwidth = 300)
